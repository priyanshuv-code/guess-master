# 🎯 Guess Master

A modern Number Guessing Game built using HTML, CSS, and JavaScript.

## 🌐 Live Demo

👉 https://guess-master-2026.netlify.app

## 🚀 Features

- Random Number Generation
- Easy / Medium / Hard Difficulty
- Timer
- Attempts Counter
- Best Score using Local Storage
- Responsive Design
- Modern UI
- Restart Game
- Input Validation

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
